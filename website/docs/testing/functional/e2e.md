# End-to-End Tests

<Lead>End-to-end tests — **manual or automated** — exercise the platform against its actual API integrations.</Lead>

This is the outermost layer of [functional testing](./index.md). No containers or mocks stand in here: whether scripted or run by hand, an E2E test goes against the real integrations.
