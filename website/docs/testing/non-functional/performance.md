# Performance Testing

<Lead>Performance tests hold the platform to its **load profiles, target throughput, and p99 latency targets**.</Lead>

Performance testing checks the platform under the load profiles it is expected to face, against the throughput it has to sustain and the p99 latency it has to stay within — that is, the response time all but the slowest 1% of requests come in under.
